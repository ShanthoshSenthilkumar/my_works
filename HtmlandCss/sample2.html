What is a website?

A website is a collection of pages on the internet that share information, provide services, or allow you to interact with content. You can access it using a web browser like Chrome or Safari by typing in its address (URL). Websites can be for shopping, learning, entertainment, or anything else you need online.


Question no:2

What is a web development?

Web development means creating and building websites for the internet. It involves designing how the website looks (frontend), making it work behind the scenes (backend), and ensuring it functions smoothly for users. Web development can include creating simple pages, complex applications, or online stores, using coding languages like HTML, CSS, and JavaScript.
